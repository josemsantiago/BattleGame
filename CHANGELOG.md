# Changelog

All notable changes to the BattleGame project will be documented in this file.

## [Unreleased]

### Added

## [1.0.0] - 2025-09-30

### Added
- CONTRIBUTING.md with comprehensive contribution guidelines
- Comprehensive test suite with 16 unit and integration tests
- MIT License file
- Detailed README.md with project documentation
- .gitignore for Python projects

### Changed
- Refactored code to use character configuration dictionary
- Improved code structure with separate functions for different game phases
- Enhanced error handling and input validation
- Updated documentation with complete feature descriptions

### Fixed
- Code quality improvements and bug fixes
- Input validation edge cases

## [0.1.0] - 2023-10-01

### Added
- Initial release of BattleGame - Dragonic Battle
- Turn-based combat system with dragon enemy
- Four playable character races (Wizard, Elf, Human, Orc)
- Character customization with name input
- Replay functionality
- Exit game option at any prompt
- Basic input validation

### Features
- Wizard: 150 damage, 70 HP
- Elf: 100 damage, 100 HP
- Human: 20 damage, 150 HP
- Orc: 95 damage, 150 HP
- Dragon: 50 damage, 300 HP
